P-CHANGE

P-CHANGE stands for "Password CHecker ANd GEnerator".

As its name suggests, this tool allows the user to :

 - check whether a given password is secure or not

 - get a new password satisfying some adjustable rules


Password Checker :

It retrieves information from http://haveibeenpwned.com and informs the user whether the provided password has already been exposed in data breaches or not.

For security purpose, the password provided by the user is never transmitted as is while retrieving information from the website mentioned above.



Password Generator :

It provides the user with a random password satisfying each selected rule and optional customization.


The rules are classified into 3 categories :

 - size : min and max number of characters

 - special characters : list of authorized special characters

 - presence : selection of at least 1 type of character (lower case letter, upper case letter, number, special character)


An optional customization is available : it allows the user to specify which characters the password must begin and/or end with.



Technical environment :

 - Programming language : Python

 - Python modules : PySide2 (Qt for Python), qtwidgets, requests, hashlib, re, random, ...

 - Version control system : Git

 - IDE : PyCharm
